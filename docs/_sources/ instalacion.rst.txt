Instalación
============

Para comenzar a usar **Edugrad**, lo primero que debes hacer es clonar el repositorio de GitHub. A continuación, te mostramos los pasos para hacerlo:

Requisitos previos
------------------

Antes de clonar el repositorio, asegúrate de tener instalado en tu sistema:

- `Git`_ para clonar el repositorio.
- `Python`_ (recomendamos Python 3.10 o superior).

.. _Git: https://git-scm.com/
.. _Python: https://www.python.org/

Pasos de instalación
--------------------

1. **Clona el repositorio**  
   Abre una terminal y ejecuta el siguiente comando para clonar el repositorio en tu máquina local:

   .. code-block:: bash

      git clone https://github.com/joseortizm/edugrad 

2. **Accede al directorio del proyecto**  
   Una vez que el repositorio se haya clonado, entra en el directorio del proyecto:

   .. code-block:: bash

      cd edugrad

3. **Instala las dependencias**  
   Numpy es suficiente sin embargo, hay dependencias adicionales que puedes instalar ejecutando:

   .. code-block:: bash

      pip install -r requirements.txt

   *(Para los ejemplos hemos utilizado bibliotecas como sklearn y matplotlib. No son parte del funcionamiento de eduGrad.)*

¡Listo para empezar!
